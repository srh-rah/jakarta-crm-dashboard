# Jakarta CRM Dashboard  

Link:

### About This Project

Analisis pengaduan warga Jakarta berbasis data CRM DKI Jakarta (2019–2020)  


---
### Dataset
| | |
|---|---|
| Source | Jakarta Open Data Portal (data.jakarta.go.id) |
| Period | 2019 – 2020 |
| Total records | 13,389 complaints |
---
### Workflow
Data Integration -> Preprocessing -> EDA -> Dashboard (Streamlit) -> Deploy
---
### Dashboard Features
- Overview metrics
- Complaint Analysis
- SKPD Performance
- Response Time Analysis
- Filter sidebar
---
## Tech Stack
Python pandas plotly streamlit
